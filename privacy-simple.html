Privacy Policy

1. Information We Collect

The TikTok Analytics Tool collects the following information:

  • TikTok API access tokens (temporarily stored in Google Apps Script)
  • Video analytics data from TikTok API
  • Basic user information provided through TikTok API

2. How We Use Information

We use the collected information to:

  • Retrieve and analyze your TikTok video data
  • Generate analytics reports
  • Provide the core functionality of the analytics tool

3. Data Storage

Data is stored securely in Google Apps Script's Properties Service and Google Spreadsheets. We do not store data on external servers.

4. Data Sharing

We do not share your personal data with third parties. Data is only used for the intended functionality of the TikTok Analytics Tool.

5. Data Security

We implement appropriate security measures to protect your data:

  • Secure API authentication
  • Encrypted data transmission
  • Limited access to stored data

6. Your Rights

You have the right to:

  • Access your stored data
  • Request deletion of your data
  • Revoke API access at any time

7. Contact Information

For privacy-related questions, please contact us through the GitHub repository.
